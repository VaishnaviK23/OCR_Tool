OCR Tool
====

This is an Optical Character Recognition Tool, made in Android Studio using Java and XML. 
The App allows the user to take a photo or an existing image of text. The text is output to a textbox in an editable format.
The user has the option to go back and take another photo. They can also save the file in .txt format.
The file can be named and saved in a directory of choice.
The output text can also be exported as a PDF.
Text-to-speech can be run on the text output for added functionality.
